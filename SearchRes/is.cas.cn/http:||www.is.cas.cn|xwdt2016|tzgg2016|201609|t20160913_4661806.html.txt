 　　2016(第七届)中国云计算学术大会 　　征文通知 　　 (2016年11月18日—11月20日，中国·长沙) 　　由中国科学技术协会、科技部、工业和信息化部指导，中国通信学会主办，中南大学承办的“2016（第七届）中国云计算学术大会(The 7th Chinese Conference on Cloud Computing，CCCC2016)”，将于2016年11月在长沙世纪金源大饭店隆重召开。 　　本次大会将围绕“云计算与大数据、透明计算与主动服务、脑科学与认知计算”等最新研究领域，为来自国内外高等院校、科研院所、企事业单位的专家、教授、学者、工程师提供一个云计算和大数据产学研用界最高水平的信息沟通平台，云计算和大数据创新成果展示与推广的渠道。本次大会将邀请10位国内外著名院士做特邀报告，40多位国内外知名专家学者、优秀论文核心团队成员、产业界首席科学做专题报告。英特尔、华为、谷歌、微软、百度、腾讯、阿里巴巴、亚信数据等知名企业专家到会做特邀报告，共同探讨云计算和大数据领域发展现状，以及所面临的关键性挑战问题和研究方向，展示各自在云计算和大数据领域的最新成果，促进我国云计算和大数据技术的研究与应用达到国际领先水平，加快建成信息强国。 　　大会诚征云计算和大数据相关领域的最新研究成果以及有关新应用与新方法的论文、案例、演示等。 　　一、  征稿范围（包括但不限于） 　　方向1：云计算和大数据的架构和基础理论 　　l  透明计算与主动服务 　　l  云计算和大数据基础设施 　　l  软件定义存储、软件定义网络、软件定义数据中心 　　l  云计算和大数据的性能改进与硬件优化 　　l  云计算、大数据、物联网和社会网络的集成平台 　　l  云计算服务迁移的最佳实现 　　l  软件定义的云计算基础理论和方法 　　l  新型大数据存储技术与平台 　　  　　方向2：面向云计算和大数据的软件工程、工具和服务 　　l  网构化软件 　　l  基于云模式和数据驱动的新型软件 　　l  平台即服务、DevOps和API管理 　　l  新型程序设计模型、质量度量、评价和管理 　　l  云计算和大数据的信息生命周期管理 　　l  云服务的业务流程和工作流管理 　　l  新型云应用系统、大数据的新理论和计算模型 　　l  基于大数据的软件智能开发方法和环境 　　  　　方向3：软件即服务、大数据分析、建模与应用 　　l  大数据分析应用与类人智能 　　l  软件即服务的业务模型、开放数据，数据即服务和决策即服务 　　l  大数据信息生命周期管理 　　l  大数据分析算法、知识发现、数据工程 　　l  大数据的视觉化分析 　　l  大规模分布式知识管理 　　  　　方向4：云计算和大数据的业务模型和应用系统 　　l  云计算和大数据在不同领域的新型业务模型和应用 　　l  API管理，API生态系统和API经济 　　l  工业互联网和分析 　　l  云端融合的感知认知与人机交互 　　l  基于数据流的大数据分析系统 　　l  面向云计算的网络化操作系统 　　l  面向特定领域的大数据管理系统 　　  　　方向5：云计算和大数据的安全、隐私、可信和质量 　　l  云计算和大数据硬件/软件的可靠性、验证与测试 　　l  云计算和大数据的可信计算和自主计算 　　l  云计算和大数据的容错 　　l  云计算和大数据的安全和隐私 　　  　　方向6：计算智能与神经科学 　　l  海量脑数据(包括脑环路和联结组学数据、高密度成像数据、大规模认知功能测量和脑疾病研究数据)分析的方法 　　l  大尺度的认知（包括记忆、决策、语言）功能脑系统计算模拟、跨层次（分子，细胞，多尺度网络，动态系统和行为）机制的分析 　　l  计算神经科学 　　l  认知系统与应用 　　l  数据挖掘 　　l  进化计算 　　l  模糊系统 　　l  粒计算 　　l  混合智能系统 　　l  图像和信号处理 　　l  神经网络 　　l  模式识别 　　l  机器人与控制应用程序 　　l  群体智能脑机接口 　　  　　方向7：大数据分析应用与类人智能 　　l  大数据知识工程基础理论及其应用研究 　　l  面向大范围场景透彻感知的视觉大数据智能分析关键技术 　　l  跨时空多源异构数据的融合、开放共享技术与平台 　　  　　方向8：云端融合的感知认识与人机交互 　　l  人机交互自然性的计算原理 　　l  云端融合的自然交互设备和工具 　　l  支持大数据理解的头戴式无障碍呈现技术 　　  　　二、投稿要求
